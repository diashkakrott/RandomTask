Описание проекта
Random Task Generator — это desktop-приложение с графическим интерфейсом на Python, которое помогает бороться с прокрастинацией и разнообразить повседневные задачи. Программа случайным образом выбирает задачу из списка, сохраняет историю всех сгенерированных заданий и позволяет фильтровать их по категориям.
Приложение идеально подходит для:
- Ежедневной рутины и саморазвития
- Тренировок и спортивных активностей
- Рабочих и учебных задач
Основные возможности
1. Генерация задач
- Нажатие на кнопку«Сгенерировать задачу»выдаёт случайное задание
- Каждая задача относится к одной из категорий: `учёба`, `спорт`, `работа`
2. Управление списком задач
- Добавление новых задач с выбором категории
- Проверка ввода — нельзя добавить пустую строку
- Очистка всей истории с подтверждением
3. История и фильтрация
- Каждая сгенерированная задача сохраняется в историю с временной меткой
-  Фильтрация истории по категориям (включая просмотр всех)
-  Автоматическое сохранение истории в JSON-файл и загрузка при запуске

Установка и запуск

Системные требования
- Python 3.7 или новее
- Библиотеки: `tkinter` (входит в стандартную поставку Python)

Инструкция по запуску
